1 What Is MIS? A management information system is an information system used for decision-making, and for the coordination, control, analysis, and visualization of information in an organization. The study of the management information systems testing people, processes and technology in an organizational context. View
2 What Different Users Can Access MIS? MIS can be accessed by the following users: School Head Masters, PCMC Admin, Commissioner, CEO, Joint-CEO, PCSCL, and School Board Authority. View
3 What Different Reports Can Be Accessed On MIS? MIS provides reports related to the following: PCMC Schools, Students, Teachers, Schools Labs Infrastructure, Trainings undertaken by Students and Teachers, Help Desk Tickets and Status, Communication between Schools and Admins, School Timetable etc. View
4 Can I Download MIS Reports? Yes, download feature is available in some specific reports available on the MIS. Reports can be downloaded in the form of spreadsheet. View
5 Can I Create New Users In MIS? Only MIS Admin can create new users in MIS. View
6 How Does “Remote Assistance For Quick Resolution” Work? Remote assistance for quick resolution helps School Admins get resolution of issues via allowing access of the impacted computer devices via remote access using Software or by explaining the issue-resolution-steps by phone calls. To allow remote access of the computer devices following software must be pre-installed. Software Name: Any Desk. Download Link: https://anydesk.com/en/downloads/windows. Software Name: Team Viewer. Download Link: https://www.teamviewer.com/en/download/windows/ View

8 How Can A User Create A Ticket? 1. User can go to the “Help Desk and Fault Logging” option on the side menu and select the “Fault Logging” option from the drop down menu.
2.User has to enter the Project, School Name and Asset from the drop-down menus.
3.The school and asset's details are filled in automatically.
4.The user then has to select the Request Type, Problem Type and Ticket Severity from the drop-down menu on the bottom of the screen.
5.Remarks can be added, to give additional information about the problem.
6.After all the details have been entered, the ticket can be registered by clicking on “Add Ticket” button.
7.The entire list of tickets generated, along with their current status can be viewed by clicking the “Asset Ticket List” button.
8.The view button in the ticket row displays all the information regarding the ticket, and also an option to change the status.		 View
